## Summary

Individual patient data (IPD) meta-analysis represents the gold standard for evidence synthesis, yet adoption remains limited due to technical barriers including software installation requirements and programming expertise. We developed IPD Meta-Analysis Pro, a browser-based application that provides comprehensive IPD meta-analysis capabilities without installation or coding requirements.

## Contribution to Scientific Literature

This manuscript addresses a significant methodological gap. While IPD meta-analysis is recognized as superior to aggregate data approaches, only 5% of published meta-analyses use IPD methods. Our tool removes the primary barriers to adoption by:

1. Eliminating software installation requirements
2. Removing the need for programming expertise
3. Providing benchmarkable statistical methods, with current-build parity claims requiring rerun validation
4. Enabling use in resource-limited settings

## Relation to Previously Published Work

Existing tools for IPD meta-analysis (R metafor, Stata ipdmetan) require substantial programming expertise. Browser-based tools exist for aggregate data meta-analysis (RevMan Web, Meta-Mar) but none provide comprehensive IPD capabilities. Our application fills this gap by combining the accessibility of browser-based tools with the methodological rigor of established statistical packages.
